### Magic Cards
-площадка для продажи игровых карточек

##### Стек технологий:

* Front: JavaScript, HBS, HTML, Bootstrap, CSS;
* Back: Node.js, Express, Sessions, Bcrypt;
* DB: PostgreSQL, Sequelize ORM.

##### Установка:

* npm ci
* npx sequelize db:create
* npx sequelize db:migrate
* npm run dev
